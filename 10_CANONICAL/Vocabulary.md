# Aspire Vocabulary

## Internal methodology terms
- Aspire Rate: growth rate of the things the user wants to buy
- Portfolio Rate: growth rate of the user's money/assets
- Aspire Gap: difference between Portfolio Rate and Aspire Rate

## User-facing language
- Aspire Rate -> Cost growth
- Portfolio Rate -> Money growth
- Aspire Gap -> Your gap

## Formula
Internal:
Aspire Gap = Portfolio Rate - Aspire Rate

User-facing:
Your gap = Money growth - Cost growth

## Product rule
Use user-facing language in:
- UI
- onboarding
- calculator
- simulator
- landing pages
- newsletter
- public content

Use internal terms ONLY in:
- technical docs
- methodology docs
- internal planning
